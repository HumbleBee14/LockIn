# LockIn

A focus app for Mac that blocks distracting websites and apps on a schedule you set once — and keeps them blocked until the time window ends.

Set it up once: pick what to block and when. When the window starts, the distractions go away across every browser and app automatically. While a session is active, there's no quick off switch — that's the point. It's built to help you get past the moment of temptation, not to be impossible to remove.

## Features

- Block websites and apps on a recurring weekly schedule
- Works across all browsers, no extension to install
- Group sites into reusable block sets, or start from a category and add your own
- Import your own lists from a file, a pasted list, or a public blocklist URL
- Resists common workarounds like changing the system clock
- Clean, native macOS interface

## Install

Install with [Homebrew](https://brew.sh):

```sh
brew tap humblebee14/lockin
brew trust humblebee14/lockin
brew install --cask lockin
```

To upgrade later:

```sh
brew upgrade --cask lockin
```

You can also download the latest signed `.dmg` directly from the [Releases page](https://github.com/HumbleBee14/LockIn/releases/latest).

## Requirements

macOS 13 or later.
